# Yaf-Server

#### 项目介绍
以 Yaf 框架为基础，进行二次完善之后专门用于服务器端开发的应用框架。该框架提供如下功能：
1）API 接口（通信、加密、应用升级、登录、注册）。
2）短信发送。
3）文件上传（本地、阿里云 OSS）。
4）日志管理（Sentry）。

该框架的目录：
1）快速搭建成熟稳定健壮的 API 服务程序。
2）百万级千万级 PV 考验。
3）成熟的文档（开发/部署/维护）。
